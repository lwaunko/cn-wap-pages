# cn-wap-pages

本仓库用于归档和发布多个独立的 HTML 页面。内容以静态网页形式存放，不绑定特定域名或网站，主要面向页面存档、展示与分发场景。

## 项目简介

`cn-wap-pages` 是一个通用型 HTML 页面仓库。仓库内每个页面均为独立、自包含的 HTML 文件，可直接在浏览器中打开，或部署至任意静态托管服务。此仓库不提供后端支持，不依赖特定平台，适合存放各类轻量级网页资源。

## 目录结构

```
cn-wap-pages/
├── pages/          # 存放所有独立 HTML 页面文件
├── assets/         # 页面引用的公共资源（CSS、JS、图片等，可选）
└── README.md       # 本文件
```

- `pages/` 目录下每个 HTML 文件为一个独立页面，文件名应反映页面内容或用途。
- `assets/` 目录用于存储多个页面共享的静态资源，若页面完全自包含可省略此目录。

## 页面归档说明

- 所有页面均为纯前端 HTML，不包含服务器端脚本。
- 页面可引用外部资源（如 CDN），但建议尽量内联或使用本地资源以保证离线可用性。
- 新增页面时，直接在 `pages/` 下添加 HTML 文件，并保持命名清晰。
- 不建议在页面内嵌入跟踪代码、广告或外部数据请求，除非明确标注。

## 使用方式

1. 直接打开 `pages/` 下的 HTML 文件即可查看页面效果。
2. 可将整个仓库克隆或下载后，通过任意静态服务器（如 Nginx、GitHub Pages 等）部署。
3. 如需引用页面，请使用原始文件链接或部署后的 URL。

## 维护说明

- 页面内容可能随时间调整，建议在引用时锁定版本或自行备份。
- 欢迎通过 Issue 提出页面相关的问题或建议，但本仓库不保证及时响应。
- 不接受涉及违法、侵权、恶意内容或商业推广的页面提交。
